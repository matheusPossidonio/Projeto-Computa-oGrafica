Questões apresentadas para desenvolvimento:

  • I) Que passos são exigidos para o desenho de um segmento de reta usando o algoritmo DDA e do ponto médio? Programe o algoritmo DDA e do ponto médio(as simetrias devem ser interpretadas tal qual visto em sala de aula) para todos os oitantes do plano cartesiano. II) Que passos são exigidos para gerar por varredura um círculo usando o método da equação explicita, método trigonométrico e do ponto médio? Programe todos. III) Que passos são exigidos para gerar uma elipse por varredura usando o método do ponto médio? Programe o algoritmo (veja apostila do curso).
  • Implemente o exercício 2.5 ( Simular o batimento do coração que se move ao longo de uma linha em uma janela na tela do computador). Vide livro Introduction to Computer Graphics Using Java 2D and 3D página 48, que está disponível no Dropbox, podem utilizar a biblioteca gráfica Java ou OpenGl, para implementar as animações propostas, leiam o capítulo 2 do livro sugerido. - KLAWONN, F. Introduction to Computer Graphics: Using Java 2D e 3D.Springer, 20083)
  • I) Implemente o algoritmo de recorte de janela de Cohen-Sutherland. ii) Implemente todas as transformações geométricas 2D (não esqueça a reflexão em torno de uma reta qualquer) e 3D. iii) Escreva um programa que se aplique uma composição de transformações 2D e 3D a um objeto visualizado na tela fora da origem do sistema cartesiano. O programa deve possibilitar o usuário definir uma determinada sequência de transformações geométricas, nas quais as composições e os parâmetros devem ser escolhidos do menu de visualização do mesmo e que possibilite exibir o objeto original em uma janela em coordenada do mundo com recorte e o objeto transformado numa view port. Observe que o algoritmo de recorte para reta deve ser aplicado na janela de trabalho do usuário e o resultado final exibido em uma viewport em coordenadas do dispositivo.
  • Faça um resumo do capítulo 21 (pagina 695 a 702) do livro Hearn, Baker and Carithers, Computer Graphics with Open GL, Fourth Edition. Implemente os seguintes exercícios: 4 e5
